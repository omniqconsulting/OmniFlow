Custom Reference Lists

⬇ Items CSV Template

Create named lists (e.g. Machine Types, Complaint Categories) whose items appear as dropdown options on ticket, checklist, and FMS forms.

📌 Where are these lists used? When creating or editing a Ticket, Checklist, or Flow Ticket, any custom field of type "Dropdown" lets you pick from one of these lists. For example: create a list called Machine Type, add items like Lathe, CNC, Press — then select "Machine Type" as the dropdown source on a ticket form field. Lists are tenant-specific and only admins can manage them here.
